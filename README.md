# MCAPrepHub
<br>

This project is a web-based platform designed for students preparing for MCA entrance examinations. <br>
The website will provide topic-wise mock tests, timed quizzes, and performance tracking features.<br>
Students will be able to register, log in, attempt tests with a countdown timer, and view their <br>
marks history along with graphical performance analysis. The system will include video lecture links <br>
for different MCA entrance topics. An admin panel will allow management of users, tests, questions, <br>
and content. The platform aims to create a real exam-like environment to improve student preparation <br>
and self-assessment.<br>
